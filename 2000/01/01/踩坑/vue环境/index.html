VUE3创建项目

以下均使用vue3：

创建项目：npm init vue@latest
进入目录之后
安装：npm install
运行：npm run dev
会在本地5173端口初始化页面

开发环境使用VSC+volar扩展

笔记

条件渲染

VUE的v-if和v-show，都是判断bool渲染。
但v-show始终会被渲染，v-if渲染切换时会被销毁重建。

如果需要频繁切换，使用v-show(但有初始渲染开销)；如果很少改变，使用v-if
